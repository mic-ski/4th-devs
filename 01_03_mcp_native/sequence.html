01_03_mcp_native / message flow

user query → agent loop → MCP & native tool calls → final response

sequenceDiagram
    autonumber

    participant App  as app.js
    participant Ag   as agent.js
    participant AI   as ai.js
    participant API  as Responses API
    participant MC   as mcp/client.js
    participant MS   as mcp/server.js
    participant NT   as native/tools.js
    participant Log  as log.js

    Note over App,Log: Startup — createMcpServer, createMcpClient, createAgent (runs once)

    App->>Ag: processQuery(query)
    Ag->>Log: logQuery(query)

    loop agent round — repeats until no tool calls, max 10

        Ag->>AI: chat({ model, tools, conversation })
        AI->>API: POST /responses
        API-->>AI: { output: [ tool_calls | message ] }
        AI-->>Ag: raw response object
        Ag->>Ag: extractToolCalls(response)

        alt tool calls present

            loop for each tool call in parallel

                Ag->>Log: logToolCall(label, name, args)

                alt MCP tool  (get_weather · get_time)
                    Ag->>MC: callMcpTool(client, name, args)
                    MC->>MS: client.callTool({ name, arguments })
                    MS->>MS: registered tool handler runs
                    MS-->>MC: { content: [{ type: text, text: JSON }] }
                    MC->>MC: JSON.parse(textContent.text)
                    MC-->>Ag: parsed result object
                else native tool  (calculate · uppercase)
                    Ag->>NT: nativeHandlers[name](args)
                    NT-->>Ag: { result: value }
                end

                Ag->>Log: logToolResult(result)

            end

            Note over Ag: tool results appended to conversation → next round

        else no tool calls
            Ag->>Ag: extractText(response)
            Ag->>Log: logResponse(text)
            Ag-->>App: final answer string
        end

    end
